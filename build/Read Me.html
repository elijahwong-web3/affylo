Affylo

Read Me and quick start guide

Install

  1. Drag Affylo.app into Applications.
  2. Open Affylo from Applications.
  3. If macOS shows an unidentified developer warning, right-click the app and choose Open once.

Use the app

  • Click Edit to enter editing mode.
  • Add companies and connections to build your structure chart.
  • Use Export PDF to save a centered PDF snapshot.
  • Use menu File → Export JSON… and Import JSON… to move charts between computers.
  • Use View → Center Chart or press Cmd+H to re-center the chart.

Local data path: ~/Library/Application Support/Affylo/data/
